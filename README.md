# AccurateBirdEyeView
This project was implemented to improve localization accuracy in soccer robot field by concatenation Inertial Measurement Unit data and camera Information. Monte carlo particle filter localization algorithm used to test this method. MPU9250 sensor module beside to implemeted madwick alghoritm on Arduino Uno board used for AHRS module.
